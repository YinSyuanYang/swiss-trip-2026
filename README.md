# 🇨🇭 瑞士之旅 2026

**家庭瑞士旅行行程規劃 App**

2026/2/7 - 2/20 ｜14天

## ✨ 功能

- 📅 **行程表** - 14天完整行程，每天可展開查看詳細時間表
- 🚂 **冰川列車** - 情人節特別行程資訊
- 💰 **預算總覽** - 交通、住宿、活動費用明細（CHF/TWD）
- 🧳 **打包清單** - 可勾選的打包清單，進度自動保存

## 🗺️ 行程亮點

- 少女峰 Top of Europe (3,454m)
- 馬特洪峰 Gornergrat (3,089m)
- 冰川列車 Glacier Express
- 列支敦士登一日遊

## 🚀 部署

### GitHub Pages

1. Push 到 GitHub
2. Settings → Pages → Branch: main → Save
3. 網址：`https://[username].github.io/swiss-trip-2026`

### Netlify

1. 拖曳 `index.html` 到 [app.netlify.com/drop](https://app.netlify.com/drop)
2. 立即獲得網址

## 📱 手機使用

- **iPhone**: Safari → 分享 → 加入主畫面
- **Android**: Chrome → 選單 → 加到主畫面

## 💾 資料儲存

打包清單的勾選狀態會自動保存在瀏覽器的 localStorage 中。
